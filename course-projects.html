5-Stage MIPS Pipeline Processor

EE126 Computer Engineering · Tufts University

Designed and implemented a five-stage pipelined MIPS processor in VHDL. The project includes core processor components such as instruction memory, data memory, ALU, register file, control logic, and pipeline registers. The design was developed and simulated on Linux using QuestaSim/ModelSim.

FPGA Thermal Side-Channel Analysis

EE156 Advanced Topics in Computer Architecture · Tufts University

Investigated whether encryption algorithms running on FPGA platforms can be identified through side-channel-related hardware characteristics. The project analyzed AES, PRESENT, Serpent, and RSA using Vivado HLS synthesis results, including latency, flip-flop usage, LUT usage, and optimization effects. A neural network classifier was trained on extracted RTL-level features to recognize different encryption algorithms.

Keywords: FPGA · Side-Channel Analysis · Encryption · Vivado HLS · Neural Network
